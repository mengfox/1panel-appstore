## 访问
安装完成后访问：
- http://服务器IP:你设置的端口（默认 9200）

## 数据目录
- `./1.0.0/data/mtab`  -> 容器内 `/app`
> 注意：请确保该目录有写入权限，否则初始化可能失败。

## MySQL / Redis 配置
安装时在 1Panel 表单中填写外部 MySQL/Redis 信息。
如果 mTab 首次进入的安装向导/后台还需要你填写一次数据库/Redis 参数，可直接复制表单里填写的值。

### 建议
- MySQL：建议 5.7+（推荐 8.x）
- Redis：按官方缓存配置文档填写（地址/端口/密码）
