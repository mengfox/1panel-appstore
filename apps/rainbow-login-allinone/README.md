# 彩虹聚合登录

## 说明
> 本包**不包含彩虹聚合登录的真实商业源码**。
> 当前内置的是一套可直接运行的 PHP 应用环境，用于在 1Panel 中完成“一体化安装、目录初始化、环境变量落地、站点可访问”的基础流程。
> 你可以在安装完成后，将真实程序源码覆盖到 `app/` 目录中继续使用。

## 默认访问
安装完成后访问：

`http://服务器IP:端口`

如果未覆盖真实业务代码，将显示一个运行状态页，用于确认：
- Nginx / OpenResty 已启动
- PHP 扩展已加载
- 可写目录权限正常

## 目录说明
- `1.0.0/app/`：内置源码骨架
- `1.0.0/nginx/`：OpenResty 配置
- `1.0.0/php/`：PHP 8.1 镜像与配置
- `1.0.0/scripts/`：初始化脚本

## 使用建议
1. 先在 1Panel 安装官方 MySQL 应用
2. 确保该 MySQL 与本应用同处 `1panel-network`
3. 导入并安装本应用
4. 在安装表单中填写 MySQL 主机、端口、库名、账号和密码
5. 将真实彩虹聚合登录源码覆盖到 `app/` 目录
