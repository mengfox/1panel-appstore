## 产品介绍

Next Terminal 是一套基于 Web 的交互审计堡垒机/运维审计系统，集成 Apache Guacamole，可在浏览器内直接访问 RDP / SSH / VNC / Telnet / Kubernetes 等资产，并提供会话审计、录像、权限策略等能力。

## 默认账户

- 默认账号：`admin`
- 默认密码：`admin`

> 登录后请第一时间修改密码，并开启 2FA/MFA（如果你有该需求）。

## 端口说明

- Web 管理端口：安装时可配置（默认 10809 → 容器内 8088）
- SSH 服务端口：安装时可配置（默认 10203 → 容器内 2022）

## 数据目录

- `./data`：Next Terminal 数据（sqlite 模式也会写入此目录）
- `./logs`：日志目录

## 注意事项

- 本应用默认使用 **sqlite** 存储数据，适合个人/小团队快速部署。
- 该镜像通常以 **amd64** 环境使用更稳定；若你是 ARM 机器，请先确认镜像是否支持你的架构。
